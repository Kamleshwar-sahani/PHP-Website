control statement Statements in php

In PHP, control statements are used to control the flow of execution in a program. They allow you to make decisions, repeat code blocks, and perform different actions based on certain conditions. Here are some commonly used control statements in PHP:

  1. Conditional Statements:
    1. if statement: Executes a block of code if a specified condition is true.
    2. if...else statement: Executes a block of code if a specified condition is true, and another block of code if the condition is false.
    3. switchstatement: Selects one of many blocks of code to be executed based on a specified value.
    4. Nested if statements in PHP allow you to have an if statement within another if statement. This allows for more complex decision-making based on multiple conditions. Here's an example of a nested if statement
  2. Looping Statements:
    1. for loop: Repeats a block of code a specific number of times.
    2. while loop: Repeats a block of code while a specified condition is true.
    3. foreach loop: Iterates over each element in an array or collection.
    4. do...while loop: Repeats a block of code while a specified condition is true, at least once.
