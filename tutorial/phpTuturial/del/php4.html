PHP superglobal variables

In PHP, superglobals are predefined variables that are accessible from any part of a PHP script. These variables are available in all scopes, including within functions, classes, and even across multiple PHP files. Super globals are useful for accessing and manipulating data that is common and shared throughout an application.

Here are some of the commonly used superglobal variables in PHP:

  1. $_GET: This variable is an associative array that contains the values of variables passed to the current script via the URL parameters (query string). For example, if the URL is example.com?name=John&age=25, you can access the values using $_GET['name'] and $_GET['age'].
  2. $_POST: This variable is an associative array that contains the values of variables sent to the current script through HTTP POST method (e.g., form submissions). The values can be accessed using $_POST['variable_name'].
  3. $_REQUEST: This variable is a combination of $_GET, $_POST, and $_COOKIE superglobals. It contains the values from both GET and POST requests, as well as cookie values. It is useful when you want to handle both GET and POST data without explicitly checking the request method.
  4. $_COOKIE: This variable is an associative array that contains any cookies sent to the current script. You can access cookie values using $_COOKIE['cookie_name'].
  5. $_SESSION: This variable is an associative array that holds session data, allowing you to store and retrieve information across multiple page requests. You need to start a session using session_start() before accessing or modifying $_SESSION values.
  6. $_SERVER: This variable is an associative array that contains information about the current HTTP request, server environment, and execution environment. It includes information such as the request method, URL, headers, server details, and more.
  7. $_FILES: This variable is an associative array that contains information about files uploaded to the server via a form with the enctype="multipart/form-data" attribute. It provides details such as the file name, temporary file location, file size, and file type.
  8. $_ENV: This variable is an associative array that contains the values of environment variables set on the server.
  9. $GLOBALS: This variable is a superglobal array that contains references to all variables available in the global scope. It allows you to access and manipulate global variables from within a function or a class.

Superglobals in PHP provide a convenient way to access and handle common data across different parts of a script. They eliminate the need for manually passing data between functions or using global variables. However, it's important to use superglobals with caution and properly validate and sanitize any user input to prevent security vulnerabilities.
